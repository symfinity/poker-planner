## Details

### 0.1.x — First public release

Planning poker rooms, hidden votes, Mercure realtime, story queue

### 0.2.x — Session and voting improvements

Guest UX, persistence options, operational troubleshooting guides

### 0.3.x — Estimation workflow

Multiple card decks, custom decks, round templates, voting presets, reveal strategies, moderator improvements, keyboard shortcuts.

### 0.4.x — Team collaboration

Observers, moderators, participant roles, avatars, reactions, presence indicators, reconnect handling, session invitations.

### 0.5.x — Project integration

Import from GitHub Issues, GitLab Issues, Jira, Azure DevOps, Linear, CSV, estimation export, webhook support.

### 0.6.x — Facilitation

Timers, break mode, discussion rounds, consensus detection, revote workflow, estimation assistants, meeting controls.

### 0.7.x — Insights and history

Session history, estimation statistics, voting distribution, confidence metrics, audit trail, reporting, analytics dashboards.

### 0.8.x — Enterprise readiness

Authentication providers, organization workspaces, permissions, private rooms, branding, retention policies, compliance settings.

### 0.9.x — API stabilization

Extension API, Mercure contracts, event contracts, configuration freeze, migration guides, documentation, cookbook.

### 1.0.x — Stable API

Semver-stable extension points, stable realtime protocol, stable configuration model, long-term compatibility guarantees.
